# Webpack 編譯範例

這是包含以下幾個籲處理器及套件的 **Webpack** 範例：

2. **scss**

3. **babel**

---

## 使用方法

```
1. $ git clone https://github.com/RexHung0302/WebpackPractice.git

2. $ cd WebpackPractice

3. $ npm start / build / build:dev / start:dev 擇一

4. Enjoy it!
```
